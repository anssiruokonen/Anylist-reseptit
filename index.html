How to Use These Recipes

  1. Click on any recipe above to view it
  2. Open the recipe page in your browser
  3. Use AnyList's import feature to import the recipe

Adding New Recipes

To add new recipes to this collection, use the command-line tool:

node add-recipe.js

The tool will automatically generate the recipe file and update this page.
